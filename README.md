# deno-intro
Deno introduction

## Backend Server:

Run the server:
```bash
 > deno run --unstable --allow-plugin --allow-read --allow-write --allow-env --allow-net deno/app.ts
``` 

## Frontend Server:

Enter the front end directory:
```bash
 > cd node-frontend-app
```

On first run you need to install dependencies:
```bash
 > npm install
```

Run the client:
```bash
 > npm start
```
